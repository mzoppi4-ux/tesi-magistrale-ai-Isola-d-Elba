# Modello rete elettrica dell'Isola d'Elba

Repository contenente codice e notebook sviluppati per il progetto di tesi magistrale.

## Obiettivo del progetto
Il lavoro riguarda la modellazione della rete elettrica dell’Isola d’Elba e l’analisi del comportamento della rete attraverso simulazioni di power flow.

## Attività svolte
- creazione del modello della rete elettrica
- utilizzo di dati reali di produzione e domanda di carico
- esecuzione di analisi di power flow
- studio di scenari evolutivi della rete
- integrazione di nuova generazione distribuita
- valutazione della hosting capacity della rete

## Contenuto del repository
- notebook Jupyter (.ipynb)
- simulazioni di rete
- analisi dati
- risultati delle simulazioni

## Software e librerie utilizzate
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
