# tesi-magistrale-ai-Isola-d-Elba
Codice e notebook sviluppati per il progetto di tesi magistrale. Il repository contiene la modellazione della rete elettrica dell’Isola d’Elba, analisi di power flow su dati reali di produzione e domanda di carico, e scenari evolutivi con integrazione di nuova generazione distribuita tramite analisi di hosting capacity.
